# [joeyqsa.dev](https://joeyqsa.dev)

My personal portfolio website showcasing projects, experience, and contact information.

## Tech Stack
- HTML
- CSS
- Hosted on GitHub Pages

## Features
- Responsive design for mobile and desktop
- SEO optimized with structured data (Schema.org)
- Accessible markup with ARIA labels

## Run Locally
Clone the repo and open `index.html` in your browser, or use a local server:

```bash
git clone https://github.com/Joeweh/Joeweh.github.io.git
cd Joeweh.github.io
npx serve .
```
